<p align="justify">
 Aspose.Imaging for .NET 是一個提供高級圖像處理功能的庫。您可以使用此 API 輕鬆創建、加載、操作、轉換、壓縮圖像或照片。也假設。圖像庫支持繪圖和使用圖形基元。圖像導出和轉換（包括統一的多頁圖像處理）是 API 的核心功能之一，以及圖像轉換（調整大小、裁剪、翻轉和旋轉、二值化、灰度、調整）、高級圖像處理功能（過濾、抖動、遮罩、去歪斜） ) 和內存優化策略。
</p>

<p align="justify">
 Aspose.Imaging for .NET 是一個靈活、穩定且功能強大的 API，能夠處理最常用的格式以及一些特殊格式，如 DjVu、DICOM、WebP 和 DNG。此外，它擴展了對 .NET 和 .NET Core 的圖像格式和處理功能的原生支持。 Aspose.Imaging 是跨平台庫，兼容 Windows 和 Linux。
</p>
