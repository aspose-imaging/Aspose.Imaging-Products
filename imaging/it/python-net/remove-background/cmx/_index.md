{{% blocks/products/pf/agp/content h2="Come rimuovere lo sfondo nei file CMX utilizzando Python" %}}

La rimozione dello sfondo da un'immagine comporta l'isolamento degli oggetti in primo piano, un'attività che richiede il riconoscimento degli oggetti. Esistono diversi approcci per identificare gli oggetti all'interno di una foto in formato CMX. Per immagini semplici con uno sfondo di colore uniforme è sufficiente un metodo automatico. Tuttavia, per le foto con figure multiple o parzialmente unite, diventa necessario pre-marcare gli oggetti. Ciò comporta la specifica di regioni rettangolari e tipi di oggetti da rimuovere. In casi complessi, l'API Cloud consente il rilevamento automatico degli oggetti. L'API Cloud fornisce un'applicazione cloud in grado di riconoscere gli oggetti all'interno delle foto, sfruttando i contorni risultanti per la rimozione dello sfondo. Dopo la rimozione, i bordi lasciati dalle figure possono essere smussati per migliorare la qualità dell'immagine. Per rimuovere lo sfondo nei file CMX, utilizzeremo [Aspose.Imaging per Python tramite .NET](/imaging/it/python-net) API che è un'API di manipolazione e conversione delle immagini ricca di funzionalità, potente e facile da usare per la piattaforma Python. Aspose.Imaging API è una soluzione di elaborazione delle immagini per creare, modificare, disegnare o convertire immagini (foto) all'interno delle applicazioni. Offre: elaborazione delle immagini multipiattaforma, incluse, a titolo esemplificativo ma non esaustivo, conversioni tra vari formati di immagine (inclusa elaborazione uniforme di immagini multipagina o multiframe), modifiche come disegnare, lavorare con primitive grafiche, trasformazioni (ridimensiona, ritaglia, capovolgi e ruota , binarizzazione, scala di grigi, regolazione), funzionalità avanzate di manipolazione delle immagini (filtro, dithering, mascheratura, raddrizzatura) e strategie di ottimizzazione della memoria. È una libreria autonoma e non dipende da alcun software per le operazioni sulle immagini. È possibile aggiungere facilmente funzionalità di conversione delle immagini ad alte prestazioni con API native all'interno dei progetti. Si tratta di API locali private al 100% e le immagini vengono elaborate sui tuoi server.
