La rimozione dello sfondo da un'immagine comporta l'isolamento degli oggetti in primo piano, un'attività che richiede il riconoscimento degli oggetti. Esistono diversi approcci per identificare gli oggetti all'interno di una foto in formato EMF. Per immagini semplici con uno sfondo di colore uniforme è sufficiente un metodo automatico. Tuttavia, per le foto con figure multiple o parzialmente unite, diventa necessario pre-marcare gli oggetti. Ciò comporta la specifica di regioni rettangolari e tipi di oggetti da rimuovere. In casi complessi, l'API Cloud consente il rilevamento automatico degli oggetti. L'API Cloud fornisce un'applicazione cloud in grado di riconoscere gli oggetti all'interno delle foto, sfruttando i contorni risultanti per la rimozione dello sfondo. Dopo la rimozione, i bordi lasciati dalle figure possono essere smussati per migliorare la qualità dell'immagine. Per rimuovere lo sfondo nei file EMF, utilizzeremo [Aspose.Imaging per Python tramite .NET](/imaging/it/python-net) API che è un'API di manipolazione e conversione delle immagini ricca di funzionalità, potente e facile da usare per la piattaforma Python.
