{{% blocks/products/pf/agp/content h2="Come cambiare lo sfondo nei file APNG usando Python" %}}

Spesso, per ottenere l'immagine ideale è necessario modificare lo sfondo. Per ottenere l'effetto immagine desiderato nel formato APNG, gli oggetti in primo piano devono essere isolati dal resto dell'immagine. Il rilevamento automatico degli oggetti è possibile se lo sfondo è uniforme. Se lo sfondo della foto non è uniforme o la separazione degli oggetti è difficile, si consiglia di premarcare l'immagine. Ciò comporta l'identificazione delle regioni rettangolari all'interno della foto in cui risiedono gli oggetti desiderati e la specificazione dei loro tipi. Questa operazione può essere eseguita manualmente o automaticamente tramite la funzionalità di riconoscimento degli oggetti dell'API Cloud. Dopo la selezione dell'oggetto e la rimozione dello sfondo originale, è possibile applicare un nuovo sfondo o implementare la trasparenza. Per cambiare lo sfondo nei file APNG, utilizzeremo [Aspose.Imaging per Python tramite .NET](/imaging/it/python-net) API che è un'API di manipolazione e conversione delle immagini ricca di funzionalità, potente e facile da usare per la piattaforma Python. Aspose.Imaging API è una soluzione di elaborazione delle immagini per creare, modificare, disegnare o convertire immagini (foto) all'interno delle applicazioni. Offre: elaborazione delle immagini multipiattaforma, incluse, a titolo esemplificativo ma non esaustivo, conversioni tra vari formati di immagine (inclusa elaborazione uniforme di immagini multipagina o multiframe), modifiche come disegnare, lavorare con primitive grafiche, trasformazioni (ridimensiona, ritaglia, capovolgi e ruota , binarizzazione, scala di grigi, regolazione), funzionalità avanzate di manipolazione delle immagini (filtro, dithering, mascheratura, raddrizzatura) e strategie di ottimizzazione della memoria. È una libreria autonoma e non dipende da alcun software per le operazioni sulle immagini. È possibile aggiungere facilmente funzionalità di conversione delle immagini ad alte prestazioni con API native all'interno dei progetti.
