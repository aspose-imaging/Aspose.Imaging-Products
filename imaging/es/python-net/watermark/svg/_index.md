La incorporación de una marca de agua es un medio excelente para que un autor mejore la visibilidad de su marca en Internet. Cuando se aplica a una imagen, una marca de agua no sólo funciona como un identificador de la fuente o del titular de los derechos de autor, sino que también constituye un instrumento promocional eficaz para el autor o la plataforma en línea. Esto se debe a que la marca de agua, ya sea el nombre del autor o un logotipo distintivo, se esparcirá junto a cada iteración digital de la imagen. Las opciones de marca de agua incluyen texto en una variedad de fuentes alrededor de la periferia de la imagen. Un logotipo semitransparente no obstaculizará la apreciación visual de la imagen o fotografía. Si el autor o el titular de los derechos de autor considera crucial resaltar su nombre, la marca de agua podría superponerse parcialmente a la imagen. Para agregar una marca de agua en imágenes SVG, utilizaremos [Aspose.Imaging for Python via .NET](/imaging/es/python-net) API que es una API de conversión y manipulación de imágenes rica en funciones, potente y fácil de usar para la plataforma Python.     {{% blocks/products/pf/agp/content h2="Acerca de Aspose.Imaging para Python API" %}}


Aspose.Imaging API es una solución de procesamiento de imágenes para crear, modificar, dibujar o convertir imágenes (fotos) dentro de las aplicaciones. Ofrece: procesamiento de imágenes multiplataforma, que incluye, entre otros, conversiones entre varios formatos de imagen (incluido el procesamiento uniforme de imágenes de varias páginas o varios cuadros), modificaciones como dibujar, trabajar con primitivas gráficas, transformaciones (redimensionar, recortar, voltear y rotar , binarización, escala de grises, ajuste), funciones avanzadas de manipulación de imágenes (filtrado, difuminado, enmascaramiento, corrección del sesgo) y estrategias de optimización de la memoria. Es una biblioteca independiente y no depende de ningún software para las operaciones de imagen. Uno puede agregar fácilmente funciones de conversión de imágenes de alto rendimiento con API nativas dentro de los proyectos. Estas son API locales 100 % privadas y las imágenes se procesan en sus servidores.
