{{% blocks/products/pf/agp/content h2="Cómo poner una marca de agua en un archivo J2K usando Python" %}}

Una marca de agua es una herramienta esencial para los autores que buscan la máxima exposición de su trabajo en Internet. Colocar una marca de agua en una imagen sirve no sólo para identificar su fuente o el titular de los derechos de autor, sino también para promocionar a su creador. Esto se debe a que la firma que contiene el nombre del autor o el nombre del recurso de Internet se distribuye junto con una copia digital de la imagen. Las marcas de agua pueden adoptar la forma de texto con una amplia gama de fuentes y pueden colocarse en el borde de la imagen. Cuando la marca de agua es transparente, no interrumpe la visualización. Sin embargo, si el autor o titular de los derechos de autor desea resaltar su nombre, la marca de agua puede superponerse parcialmente a la imagen. Para poner una marca de agua en el archivo de imagen J2K, usaremos [Aspose.Imaging for Python via .NET](/imaging/es/python-net) API que es una API de conversión y manipulación de imágenes rica en funciones, potente y fácil de usar para la plataforma Python. Puede instalarlo usando el siguiente comando desde el comando de su sistema.

    {{% blocks/products/pf/agp/content h2="Acerca de Aspose.Imaging para Python API" %}}


Aspose.Imaging API es una solución de procesamiento de imágenes para crear, modificar, dibujar o convertir imágenes (fotos) dentro de las aplicaciones. Ofrece: procesamiento de imágenes multiplataforma, que incluye, entre otros, conversiones entre varios formatos de imagen (incluido el procesamiento uniforme de imágenes de varias páginas o varios cuadros), modificaciones como dibujar, trabajar con primitivas gráficas, transformaciones (redimensionar, recortar, voltear y rotar , binarización, escala de grises, ajuste), funciones avanzadas de manipulación de imágenes (filtrado, difuminado, enmascaramiento, corrección del sesgo) y estrategias de optimización de la memoria. Es una biblioteca independiente y no depende de ningún software para las operaciones de imagen. Uno puede agregar fácilmente funciones de conversión de imágenes de alto rendimiento con API nativas dentro de los proyectos. Estas son API locales 100 % privadas y las imágenes se procesan en sus servidores.
