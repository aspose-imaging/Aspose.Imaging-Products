{{% blocks/products/pf/feature-page-section  h2="Aspose.Imaging .NET redimensionar solo licencias" %}}
¿Le preocupa solo el cambio de tamaño de la imagen? La licencia de Aspose.Imaging Resize puede ayudarlo a implementar el cambio de tamaño requerido mediante programación sin necesidad de licenciar otras características del producto. <br/>
<i><b>Para hacerlo realidad, puede:</b></i>
<ul>
<li>
Compre la licencia apropiada en <a href="https://purchase.aspose.market/">https://purchase.aspose.market/</a>
</li>
<li>
Agregue <a href="https://www.nuget.org/packages/Aspose.Imaging">referencia de nuget a Aspose.Imaging .NET</a> a partir de v23.2 para su proyecto;
</li>
<li>
Implemente el muestreo hacia abajo o hacia arriba de la imagen requerido utilizando uno o varios de los <a href="https://reference.aspose.com/imaging/net/aspose.imaging/resizetype/">algoritmos de cambio de tamaño</a> admitidos
</li>
<li>
Utilice la licencia adquirida para <a href="https://docs.aspose.com/imaging/net/licensing/">establecer la licencia</a>.
</li>
</ul>
<i><b>¿Qué plataformas .NET son compatibles?</b></i> <br/>
La licencia de cambio de tamaño de imagen funciona para NETStandard2.0 y plataformas .NET superiores;<br/>
<i><b>¿Qué se espera cuando se utilizan entradas de API que no tienen licencia de cambio de tamaño comprada?</b></i><br/>
En tal caso, se utilizará el modo de prueba.<br/>
<i><b>¿Dónde puedo encontrar qué entradas de API son adecuadas para mi licencia de cambio de tamaño de imagen?</b></i><br/>
Revise los fragmentos de código disponibles en las páginas secundarias para entender el punto.<br/>
<i><b>¿Hay una demostración en vivo para ver cómo funciona?</b></i><br/>
Hay una <a href="https://products.aspose.app/imaging/es/image-resize/">aplicación de cambio de tamaño de imagen Aspose.Imaging</a> , que puede probar antes de comprar la licencia de cambio de tamaño. <br/>
