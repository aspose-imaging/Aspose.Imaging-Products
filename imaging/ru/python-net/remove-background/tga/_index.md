Удаление фона изображения подразумевает точную идентификацию значимых объектов. Для изображений в формате TGA существует несколько способов определения объектов. В относительно простых случаях автоматический метод прекрасно справляется с изображениями, где фон равномерен. Однако, на фотографиях, содержащих несколько фигур, или при слиянии объектов с фоном, целесообразно использовать предварительное обозначение объектов. Это включает предварительную ручную разметку прямоугольных участков и указание типов объектов, которые необходимо выделить. В более сложных случаях автоматического выделения объектов можно использовать Cloud API. Cloud API предоставляет из себя облачное приложение, распознающее объекты на фотографии и применяющее полученные контуры для удаления фона. После удаления фона, размытие краев оставшихся объектов может улучшить качество финального изображения.
