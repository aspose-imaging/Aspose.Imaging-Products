Часто при работе с изображением PNG требуется поменять его фон. Для выделения областей относящихся к фону, первым шагом является определения объектов на переднем плане фотографии, и их отделение от остальной части изображения. Если фон представляет собой равномерную область, формирование контуров объектов возможно автоматически. В случае неравномерного фона на фотографии или сложностей с отделением желаемого объекта от заднего плана, рекомендуется использовать метод предварительной разметки изображения. Следует выделить прямоугольные области на фотографии, где располагаются ожидаемые объекты, и указать их тип. Это действие можно выполнить вручную или воспользоваться Cloud API для автоматического определения объектов. После выделения объектов и удаления исходного фона можно установить новый фон или сделать его прозрачным. Для изменения фона изображения в формате PNG мы воспользуемся [Aspose.Imaging for Python via .NET](/imaging/ru/python-net) API, который представляет собой многофункциональный, мощный и простой в использовании API для обработки и преобразования изображений для платформы Python.
