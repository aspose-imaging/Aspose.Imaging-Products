Водяний знак — це важливий інструмент для авторів, які прагнуть отримати максимальну доступність своєї роботи в Інтернеті. Розміщення водяного знака на зображенні служить не тільки для ідентифікації його джерела або власника авторських прав, але й для реклами його творця. Це пов’язано з тим, що підпис із ім’ям автора чи назвою інтернет-ресурсу поширюється разом із цифровою копією зображення. Водяні знаки можуть мати форму тексту з широким діапазоном шрифтів і можуть розташовуватися на краю зображення. Коли водяний знак прозорий, він не заважає перегляду. Проте, якщо автор або власник авторських прав бажає підкреслити своє ім’я, водяний знак може частково перекривати зображення. Щоб додати водяний знак у файл зображення PNG, ми використаємо [Aspose.Imaging for Java](https://products.aspose.com/imaging/java) API, який є багатофункціональним, потужним і простим у використанні API для обробки зображень і перетворення для платформи Java. Ви можете завантажити його останню версію безпосередньо з [Maven](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-imaging) і встановити на своєму Maven на основі проекту, додавши такі конфігурації до pom.xml.

Це 100% приватні локальні API, а зображення обробляються на ваших серверах.

PNG, Portable Network Graphics, відноситься до типу формату растрового зображення, який використовує стиснення без втрат. Цей формат файлу було створено як заміну формату Graphics Interchange Format (GIF) і не має обмежень щодо авторських прав. Однак формат файлу PNG не підтримує анімацію. Формат файлу PNG підтримує стиснення зображень без втрат, що робить його популярним серед користувачів. З плином часу PNG перетворився на один із найбільш використовуваних форматів файлів зображень. Майже всі операційні системи підтримують відкриття файлів PNG. Наприклад, засіб перегляду Microsoft Windows має можливість відкривати файли PNG, оскільки ОС за замовчуванням підтримує підтримку, доступну як частину встановлення.
