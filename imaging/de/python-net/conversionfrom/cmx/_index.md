<p align=justify>Mit Aspose können Sie mithilfe von Aspose.Imaging für Python über .NET automatisch zwischen dem CMX-Format und anderen bereitgestellten Formaten umwandeln. Dateien mit der Erweiterung CMX sind Corel Exchange-Bilddateiformate, die von CorelSuite-Anwendungen als Präsentation verwendet werden. Sie enthält Bilddaten als Vektorgrafik sowie Metadaten, die das Bild beschreiben. CMX-Dateien können mit CorelDraw, Corel Presentations, Paint Shop Pro und einigen Versionen von Adobe Illustrator geöffnet werden.</p>
