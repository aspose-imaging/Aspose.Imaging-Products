<p align=justify>Mit Aspose können Sie mithilfe von Aspose.Imaging für Python über .NET automatisch zwischen dem ICO-Format und anderen bereitgestellten Formaten umwandeln. Das ICO-Dateiformat ist ein Bilddateiformat für Computersymbole in Microsoft Windows. ICO-Dateien enthalten ein oder mehrere kleine Bilder in mehreren Größen und Farbtiefen, sodass sie entsprechend skaliert werden können. In Windows müssen alle ausführbaren Dateien, die dem Benutzer auf dem Desktop, im Startmenü oder im Windows Explorer ein Symbol anzeigen, das Symbol im ICO-Format enthalten.</p>
