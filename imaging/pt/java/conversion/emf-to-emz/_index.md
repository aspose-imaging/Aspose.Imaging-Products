Você tem a opção de salvar a imagem em um formato descompactado para obter qualidade ideal ou convertê-la em um formato compactado sem perdas para reduzir o tamanho do arquivo. Algoritmos especialmente projetados para compactação de dados de imagem permitem uma redução significativa no tamanho do arquivo, preservando ao mesmo tempo uma qualidade de imagem aceitável.
