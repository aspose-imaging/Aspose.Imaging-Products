Začlenění vodoznaku je pro autora vynikajícím prostředkem, jak zvýšit viditelnost své značky v internetovém prostoru. Když je vodoznak aplikován na obrázek, funguje nejen jako identifikátor zdroje nebo držitele autorských práv, ale také jako účinný propagační nástroj pro autora nebo online platformu. Vodoznak, ať už jde o jméno autora nebo výrazné logo, bude totiž roztroušeno vedle každé digitální iterace obrázku. Možnosti vodoznaku zahrnují text v různých typech písem po obvodu obrazu. Poloprůhledné logo nebude bránit vizuálnímu zhodnocení obrázku nebo fotografie. Pokud by autor nebo držitel autorských práv považoval za důležité zdůraznit své jméno, vodoznak může částečně překrýt obrázek. K přidání vodoznaku do obrázků ve formátu J2K použijeme [Aspose.Imaging pro Python přes .NET](/imaging/cs/python-net) API, které je funkčně bohaté, výkonné a snadno použitelné rozhraní API pro manipulaci a konverzi obrázků pro platformu Python. Aspose.Imaging API je řešení pro zpracování obrázků pro vytváření, úpravu, kreslení nebo konverzi obrázků (fotografií) v rámci aplikací. Nabízí: multiplatformní zpracování obrazu, mimo jiné včetně převodů mezi různými formáty obrázků (včetně jednotného vícestránkového nebo vícesnímkového zpracování obrazu), úpravy jako kreslení, práci s grafickými primitivy, transformace (změna velikosti, oříznutí, převrácení a otočení). binarizace, stupně šedi, úprava), pokročilé funkce pro manipulaci s obrázky (filtrování, rozklad, maskování, vyrovnání sklonu) a strategie optimalizace paměti. Je to samostatná knihovna a není závislá na žádném softwaru pro operace s obrázky. V rámci projektů lze snadno přidat vysoce výkonné funkce pro konverzi obrázků s nativními rozhraními API. Jedná se o 100% soukromá on-premise API a obrázky se zpracovávají na vašich serverech.
