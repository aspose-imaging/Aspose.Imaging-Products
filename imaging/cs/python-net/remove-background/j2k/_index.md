Odstranění pozadí z obrázku zahrnuje izolaci objektů v popředí, což je úkol, který vyžaduje rozpoznání objektu. Existuje několik přístupů k identifikaci objektů na fotografii ve formátu J2K. Pro jednoduché obrázky s jednotným barevným pozadím postačí automatická metoda. U fotografií s více nebo částečně sloučenými postavami je však nutné předem označit objekty. To zahrnuje specifikaci obdélníkových oblastí a typů objektů pro odstranění. Ve složitých případech umožňuje Cloud API automatickou detekci objektů. Cloud API poskytuje cloudovou aplikaci schopnou rozpoznávat objekty na fotografiích a využívat výsledné obrysy pro odstranění pozadí. Po odstranění lze okraje zanechané postavami vyhlazovat, aby se zvýšila kvalita obrazu. K odstranění pozadí v souborech J2K použijeme [Aspose.Imaging pro Python přes .NET](/imaging/cs/python-net) API, které je funkčně bohaté, výkonné a snadno použitelné rozhraní API pro manipulaci a konverzi obrázků pro platformu Python.
