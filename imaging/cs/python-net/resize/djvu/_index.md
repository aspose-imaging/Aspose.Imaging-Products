Často vzniká potřeba změnit jejich rozměry a připravit obrázky nebo fotografie v různém rozlišení. Například stránky s galeriemi obrázků s náhledy vyžadují malé soubory miniatur, zatímco stránky s hlavními obrázky vyžadují verze s vysokým rozlišením. Během změny velikosti obrázku můžete také zvolit stupeň komprese dat, abyste dosáhli rovnováhy mezi velikostí a kvalitou obrázku. Menší velikosti obrázků vedou k rychlejšímu načítání z internetu, což zlepšuje uživatelský zážitek z procházení vašich webových stránek. Aspose.Imaging API je řešení pro zpracování obrázků pro vytváření, úpravu, kreslení nebo konverzi obrázků (fotografií) v rámci aplikací. Je to samostatná knihovna a není závislá na žádném softwaru pro operace s obrázky. V rámci projektů lze snadno přidat vysoce výkonné funkce pro konverzi obrázků s nativními rozhraními API. Jedná se o 100% soukromá on-premise API a obrázky se zpracovávají na vašich serverech.
