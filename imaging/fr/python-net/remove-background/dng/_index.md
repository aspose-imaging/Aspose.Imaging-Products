{{% blocks/products/pf/agp/content h2="Comment supprimer l'arrière-plan des images et des photos DNG avec Python" %}}

Supprimer l’arrière-plan d’une image ou d’une photo nécessite une identification précise des objets proéminents. Pour les images DNG, différentes méthodes sont disponibles pour la définition des objets. Dans des scénarios simples, l’approche automatisée gère efficacement les images avec des arrière-plans uniformes. Néanmoins, lorsqu'il s'agit de photos présentant plusieurs personnages ou objets fusionnant avec l'arrière-plan, il est recommandé de procéder à une désignation préliminaire de l'objet. Cela implique de délimiter manuellement des zones rectangulaires et de spécifier les types d'objets à mettre en évidence. Dans les cas plus complexes d’allocation automatisée d’objets, l’API Cloud constitue une alternative. Cette application basée sur le cloud identifie les objets dans une photo et utilise les contours résultants pour éliminer l'arrière-plan. Après la suppression de l’arrière-plan, l’amélioration des bords des objets restants peut améliorer considérablement la qualité globale de l’image. Pour supprimer l'arrière-plan des fichiers DNG, il est suggéré d'utiliser le [Aspose.Imaging for Python via .NET](/imaging/fr/python-net) API qui est une API de manipulation et de conversion d'images riche en fonctionnalités, puissante et facile à utiliser pour la plate-forme Python. Vous pouvez l'installer en utilisant la commande suivante à partir de votre commande système.
