<p align=justify>À l'aide d'Aspose, vous pouvez effectuer automatiquement une transformation entre le format DIB et d'autres formats fournis à l'aide d'Aspose.Imaging pour Python via .NET. Un fichier DIB (Device Independent Bitmap) est un fichier d'image raster dont la structure est similaire à celle des fichiers Bitmap standard (BMP), mais dont l'en-tête est différent. Il peut être ouvert avec presque toutes les applications capables d'ouvrir un fichier BMP standard sous Windows ainsi que macOS. DIB sont des fichiers binaires et ont un format de fichier complexe similaire à BMP.</p>
