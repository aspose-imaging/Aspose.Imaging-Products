{{% blocks/products/pf/feature-page-section  h2="Aspose สามารถแปลงรูปภาพ WMF เป็นรูปแบบรูปภาพที่หลากหลายโดยใช้ Aspose.Imaging สำหรับ Python ผ่าน .NET" %}}
<p align=justify>เมื่อใช้ Aspose คุณจะแปลงระหว่างรูปแบบ WMF กับรูปแบบอื่นๆ ที่ให้มาโดยอัตโนมัติได้โดยใช้ Aspose.Imaging สำหรับ Python ผ่าน .NET ไฟล์ที่มีนามสกุล WMF แสดงถึง Microsoft Windows Metafile (WMF) สำหรับการจัดเก็บข้อมูลเวกเตอร์และรูปแบบบิตแมป เพื่อให้แม่นยำยิ่งขึ้น WMF อยู่ในหมวดหมู่รูปแบบไฟล์เวกเตอร์ของรูปแบบไฟล์กราฟิกที่ไม่ขึ้นกับอุปกรณ์ Windows Graphical Device Interface (GDI) ใช้ฟังก์ชันที่จัดเก็บไว้ในไฟล์ WMF เพื่อแสดงภาพบนหน้าจอ WMF เวอร์ชันที่ได้รับการปรับปรุงมากขึ้น หรือที่เรียกว่า Enhanced Meta Files (EMF) ได้รับการเผยแพร่ในภายหลัง ซึ่งทำให้รูปแบบมีคุณลักษณะที่สมบูรณ์ยิ่งขึ้น ในทางปฏิบัติ WMF จะคล้ายกับ SVG</p>
