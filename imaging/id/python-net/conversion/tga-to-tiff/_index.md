Gambar yang dibuat dalam editor grafis kemungkinan besar dalam format vektor. Anda memiliki opsi untuk mengonversi gambar dalam format tidak terkompresi untuk kualitas unggul atau menyimpannya ke format terkompresi lossless untuk meminimalkan ukuran file.
