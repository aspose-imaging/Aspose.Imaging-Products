Gambar yang dibuat dalam editor grafis kemungkinan besar dalam format vektor. Dalam kasus seperti itu, untuk publikasi situs web, situs tersebut harus menjalani rasterisasi dan disimpan dalam format raster. Anda memiliki opsi untuk mengonversi gambar dalam format tidak terkompresi untuk kualitas unggul atau menyimpannya ke format terkompresi lossless untuk meminimalkan ukuran file.
